# Anti-Patterns

- Falar de IA de forma abstrata e vazia
- Usar jargao financeiro pesado para um publico amplo
- Criar CTA generico como "saiba mais"
- Fazer promessa exagerada ou nao verificavel
- Produzir carrossel com slide 1 fraco
- Encher a arte de texto pequeno demais
- Publicar sem confirmacao explicita do usuario
- Silenciar falhas de dry-run ou de API
