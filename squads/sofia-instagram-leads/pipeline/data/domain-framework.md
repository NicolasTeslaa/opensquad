# Domain Framework

## Strategic Model

1. Identify a painful and common financial problem.
2. Reframe it in plain language with one clear promise.
3. Teach a practical shift the audience can apply quickly.
4. Connect that shift to Sofia's value proposition.
5. End with a CTA that moves the reader to the next interaction.

## Approved Content Angles

- Problema -> Solucao
- Mito vs Realidade
- Tutorial rapido
- Lista de erros
- Checklist de diagnostico

## Lead Generation Rules

- O post deve resolver uma dor concreta, nao falar apenas de funcionalidades.
- A promessa deve ser simples e acionavel.
- O CTA deve pedir uma unica acao.
- Toda afirmacao importante precisa soar crivel e especifica.
- O conteudo deve ser util mesmo para quem ainda nao esta pronto para comprar.
